A GUI-based Python utility engineered to remove image backgrounds using a suite of AI models. It streamlines testing by processing one image with multiple algorithms (rembg, mediapipe) and saving separate, transparent outputs for easy side-by-side comparison.

A cross-platform Python GUI utility that converts structured Excel spreadsheets into standard Netscape HTML bookmark files. It automatically detects nested folder hierarchies from spreadsheet columns and generates a universal file compatible for import into Chrome, Firefox, Safari, and Edge.
