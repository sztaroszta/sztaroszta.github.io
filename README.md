**Welcome to my digital workspace**

[starosta.app](https://starosta.app/) is an active R&D environment and the primary deployment platform for my work in **product management, business intelligence, and software engineering.**

I built it as an interactive hub to demonstrate how I structure information, solve problems, and connect the dots between **data architecture, commercial strategy, and technical execution.**

The centralized application currently houses my open-source tools, applied research, latest publications, and milestone logs.

## ðŸ§  About The Architecture

This portfolio is engineered natively using **Vanilla HTML, CSS, and JavaScript**. By omitting external frontend frameworks, the architecture maintains zero dependency overhead, enabling lightning-fast load times and absolute control over the custom UI/UX interactions.

### Core Features:

- **Custom Routing Engine:** A bespoke hash-based navigation system that handles view switching (`#card`, `#matrix`, `#gallery`, `#analytics`, `#about`) without page reloads.
- **Multi-Dimensional Views:**
  - **Overview (Cards):** Expandable/collapsible bento-box layouts.
  - **Registry (Matrix):** A complex, filterable, and sortable data table with custom export capabilities (CSV, JSON, Image).
  - **Visuals (Gallery):** A high-performance media grid with custom masonry, dynamic scaling, and an integrated lightbox.
  - **Analytics (Dashboard):** Interactive 3D CSS transformations featuring an Orbit and Carousel view for data visualization.
  - **Strategic Profile (About):** A dedicated layout detailing my professional mandate, strategic stack, and core protocols.
- **Advanced State Management:** Maintains complex filter, sort, and layout states across different views.
- **Premium UI/UX:** Features glassmorphism (frosted glass), custom cursors, contextual tooltips, and fluid CSS animations.
